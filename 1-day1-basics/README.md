## üéÆ How to Get Started

### For Students:
1. **You'll be typing the actual code!** - The file has guides but you fill in the details
2. Open the `starter.js` file - this has TODO items for you to complete
3. **The HINTS show you exactly what to type** - but you have to type it yourself!
4. Follow the STEP-BY-STEP comments and complete each TODO
5. **Test after each step!** Run your code to see what changes
6. Try the "TRY THIS" suggestions - they're fun modifications!
7. Don't worry if you make mistakes - that's how you learn!

### Quick Start:
1. Go to [p5.js Web Editor](https://editor.p5js.org/)
2. Copy your code from `starter.js` into the editor
3. Click the Play button ‚ñ∂Ô∏è to see your game!
4. Use arrow keys or WASD to move around

### For Teachers

- **Students type all the actual code** - `starter.js` has TODOs with detailed HINTS

- **Each TODO can be completed independently** - Students can work at their own pace

## üí° Easy Modifications to Try

**Colors** (change the numbers in `fill()`):
- `fill(255, 0, 0)` = Red
- `fill(0, 255, 0)` = Green  
- `fill(255, 0, 255)` = Purple
- `fill(255, 255, 0)` = Yellow

**Size & Speed**:
- Make `player.size` bigger (30) or smaller (10)
- Make `player.speed` faster (8) or slower (2)
- Change canvas size: `createCanvas(1000, 700)`

**Text**:
- Change the game title to their name
- Add their own instructions
