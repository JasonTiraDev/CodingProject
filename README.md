## About This Project

Mini Survivors is a 6-day JavaScript game development curriculum designed for teaching Boy Scouts (ages 14-16) intro to coding using p5.js. Students build a complete survival game from scratch, learning fundamental programming concepts through hands-on game development.

### What Students Learn:
- **Day 1** - Player movement and boundaries
- **Day 2** - Combat system with shooting mechanics
- **Day 3** - Complete game with power-ups and UI
- **Bonus Days 4-6** - Visual effects, audio, and advanced features
